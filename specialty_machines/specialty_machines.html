Satisfactory Modeler

Specialty Machines

In The Modeler you will find all the recipes that are in the game in addition to some specialty machines.


Outpost

Outposts are a visual separation to help you organize your factory. They behave somewhat like a folder on a computer, you can open one up and go inside it. You can nest any number of outposts inside other outposts.
Outposts can also have inputs and outputs to connect what is inside to what is outside. They can be created by dropping connections onto them from the outside, or by using the plus button inside, or dropping connections onto the plus button.


Blueprint

Blueprints are like outposts, but the things inside can be duplicated when calculating. Be sure to always add a limit to the inside of a blueprint so it can define what a single copy is. The calculated value of that blueprint is how many copies of it you will need to build.


Splurger

This can be used to represent a splitter or a merger. In the Modeler, all connection points act as splitters or mergers by default for convenience, so there is often no need for a Splurger, however there are times you may need one or want want for aesthetics. A Splurger’s input connection point can act as a merger and its output can act as a splitter.


Priority Splurger

This can be used to represent a Smart Splitter, Programmable Splitter, Priority Merger, or even properly set up pipes. It comes with two inputs and two outputs. The top is the top priority and will always take from or put to it first. The bottom is bottom priority and will only be used if the top does not supply or accept everything.
Note that the priority aspect of this is only used in the Full calculator.

This comes in three flavors for aesthetics:

  • Priority Splurger: Can be used as both a priority splitter and/or a priority merger.
  • Priority Splitter: Can only be used as a priority splitter.
  • Priority Merger: Can only be used as a priority merger.

AWESOME Sink

Just like the awesome sink in game. Because the number of items per minute it can consume depends on belt speed, you can set the belt speed for it to be able to calculate when not in parts per minute mode.


Storage Container

This can do more than represent a storage container in the game, you can also use them to represent more abstract concepts.

The storage container comes in several different modes for calculation (Note these modes mostly only matter when using the full calculator):

  • Partially Full: Also Partially Empty. Extra parts can be either added or taken from this. This means that it will neither back up nor be short on parts.
  • Full: Extra parts can be taken from this, but more parts cannot be added than are taken. This means that it will never be short on parts but can back up.
  • Empty: Extra parts can be added to this, but more parts cannot be taken than are added. This means that it will never back up but can be short on parts.
  • Input = Output: The input and output must match exactly. This makes it the same as using a splurger but with a different icon.

Dimensional Depot

Simply represents a Dimensional Depot in the game. It can be set to different consumption rates when calculating when it is not in parts per minute mode.


Space Elevator Phases

You can find the Space Elevator Phases with the list of all recipes, these represent the resources need to complete each space elevator phase.
