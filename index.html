Satisfactory Modeler

Satisfactory Modeler is a visual calculator that allows quick planning and calculations for any custom layout.
Place recipes, connect them together, add limits and it will tell you how many machines you need to build.
Satisfactory Modeler is NOT designed to be a layout planner detailing exactly where and in what orientation to place things, but rather a more abstract visual calculator to help you plan how many machines you need.
